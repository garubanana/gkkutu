가루끄투에 오신 것을 진심으로 환영합니다!

가루끄투는 게스트 게임 제한 서버입니다.(관전만 가능) 소셜(social) 로그인을 꼭 사용 해 주세요!

네이버 로그인은 아직 사용 할 수 없으니 구글, 페이스북 로그인을 사용해 주십시오.

가루끄투 공식 카페 바로가기

우측 상단의 디스코드 방에서 모든 공지와 단어 추가 내용이 올라갑니다.

html5, javascript를 이용해 개발 되었기 때문에 크롬을 사용해서 접속하는 것을 권장합니다.

현재 추가되어 있는 주제는, 이나즈마 일레븐, 한국 노래, 동방 프로젝트, 클래시 로얄/오브 클랜, 읍/면/동, 동요, 속담, 언더테일, 파워 쿵쿵따, 사자성어, 자동차, 아이작의 번제, 워킹데드, 명화, 법률 등이 있습니다! 우리 서버는 분쟁, 욕설, 핵 사용 등을 철저하게 단속하고 있으니 이점 참고하시기 바랍니다.
